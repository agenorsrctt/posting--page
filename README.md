Blog DevStart

Landing page interativa de blog desenvolvida com HTML, CSS e JavaScript, simulando a criação e exibição de postagens com consumo de API.

--


#Estrutura
Barra de navegação (nav)
Seção:
Meu Blog (descrição)
Criar novo post (formulário)
Últimas postagens (lista dinâmica)
Rodapé com informações do autor

--

#Funcionalidades
Criação de novos posts
Listagem de posts vindos da API
Atualização dinâmica do conteúdo na tela
Envio de dados via fetch
Manipulação do DOM com JavaScript

--

#Tecnologias
HTML5
CSS3
JavaScript (Vanilla)
Fetch API

--


#Integração
API pública:
https://jsonplaceholder.typicode.com

--


#Execução

Abrir o arquivo index.html no navegador.
